##############
 Installation
##############

The most recent release can be installed from `PyPI
<https://pypi.org/project/umls-downloader>`_ with uv:

.. code-block:: console

    $ uv pip install umls-downloader

or with pip:

.. code-block:: console

    $ python3 -m pip install umls-downloader

*********************
 Installing from git
*********************

The most recent code and data can be installed directly from GitHub with uv:

.. code-block:: console

    $ uv pip install git+https://github.com/cthoyt/umls-downloader.git

or with pip:

.. code-block:: console

    $ python3 -m pip install git+https://github.com/cthoyt/umls-downloader.git

****************************
 Installing for development
****************************

To install in development mode with uv:

.. code-block:: console

    $ git clone git+https://github.com/cthoyt/umls-downloader.git
    $ cd umls-downloader
    $ uv pip install -e .

or with pip:

.. code-block:: console

    $ python3 -m pip install -e .
